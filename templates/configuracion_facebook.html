¿Cómo obtener las credenciales?
1 Crear App de Facebook

Ve a Facebook Developers y crea una nueva aplicación.

2 Configurar Marketing API

Agrega el producto "Marketing API" a tu aplicación.

3 Obtener Credenciales

Copia el App ID y App Secret desde la configuración básica.

4 Generar Access Token

Usa el Graph API Explorer para generar un token de larga duración.

Importante: Asegúrate de que tu aplicación tenga los permisos ads_management y ads_read.
